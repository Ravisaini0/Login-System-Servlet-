# 🚀 CRUD Application (Login & Registration System)

A full-stack web application built using **Java Servlets, JSP, and MySQL**.  
This project implements secure user authentication and complete CRUD operations for user management.

---

## 📌 Features

- 🔐 User Registration & Login System
- ✅ Secure Authentication
- 📝 CRUD Operations (Create, Read, Update, Delete)
- 🗄️ MySQL Database Integration
- 🔄 Session Management
- 🚪 Logout Functionality

---

## 🛠️ Tech Stack

- 💻 Java (Servlets)
- 🌐 JSP (Java Server Pages)
- 🗄️ MySQL
- 🔌 JDBC
- 🎨 HTML, CSS (Frontend)
